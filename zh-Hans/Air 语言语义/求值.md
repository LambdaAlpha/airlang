# 求值

## 上下文

### 单元格

- `.value`：表示单元格中的值

### 配对

- `.left`：表示配对的左元素
- `.right`：表示配对的右元素

可读写

### 列表

- `.first`：表示列表的第一个元素
- `.last`：表示列表的最后一个元素
- 整数：表示列表第 `i` 个元素

可读写

### 映射

键 `k` 表示映射中 `k` 所映射的值

可读写，可新增，可移除

### 引用

- `.source`：表示引用中的源

### 调用

- `.function`：表示调用的函数
- `.input`：表示调用的输入

可读写

## 键

- `_a` ➔ `a`
- `.a` ➔ `.a`
- `a` ➔ `v`，`v` 为上下文中键 `a` 所绑定的值

## 引用

`_(v)` ➔ `v`

## 调用

`_ f i` ➔ `f'(i')`，其中 `x'` 表示 `x` 求值的结果，下同。

## 结构

- `.(v)` ➔ `.(v')`
- `v1 : v2` ➔ `v1' : v2'`
- `[v1, v2, ..., vn]` ➔ `[v1', v2', ..., vn']`
- `{k1 : v1, k2 : v2, ..., kn : vn}` ➔ `{k1 : v1', k2 : v2', kn : vn'}`

## 其他

其他值的求值规则为 `v` ➔ `v`。
