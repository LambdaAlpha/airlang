# 函数

## _function.new

类型：function

上下文访问：无关

构造一个函数。

i 是一个映射

- id：函数的 id
- code：函数调用过程的表示，`(ctx : input) : body`，`ctx` 为上下文变量名，`input` 为输入变量名，`body` 为函数体
- context：函数内上下文
- context_access：上下文访问权限
  - free：无关
  - constant：不变
  - mutable：可变

## _function.represent

类型：function

上下文访问：无关

表示一个函数。输出一个映射

- id：函数的 id
- raw_input：是否不对输入求值
- code：函数调用过程的表示，`(ctx : input) : body`，`ctx` 为上下文变量名，`input` 为输入变量名，`body` 为函数体
- context：函数内上下文
- context_access：上下文访问权限
  - free：无关
  - constant：不变
  - mutable：可变

## _function.apply

类型：function

上下文访问：可变

若 i 为 `func : input`，则以 input 为输入调用函数 func。

## _function.context_access

类型：function

上下文访问：不变

输出函数 c 的上下文访问权限。

- free：无关
- constant：不变
- mutable：可变

## _function.raw_input

类型：function

上下文访问：不变

输出函数 c 是否不对输入求值。

## _function.is_primitive

类型：function

上下文访问：不变

输出函数 c 是否为基础函数。

## _function.id

类型：function

上下文访问：不变

输出函数 c 的 id。

## _function.code

类型：function

上下文访问：不变

输出组合函数 c 的代码的表示。

## _function.context

类型：function

上下文访问：不变

输出组合函数 c 的内上下文。
