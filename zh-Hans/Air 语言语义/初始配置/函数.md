# 函数

## _function.new

类型：function

上下文访问：无关

构造一个函数。

i 是一个映射

- code：函数调用过程的表示，`(ctx : input) : body`，`ctx` 为上下文变量名，`input` 为输入变量名，`body` 为函数体
- context：函数内上下文
- context_access：上下文访问权限
  - `_free`：无关
  - `_constant`：不变
  - `_mutable`：可变

## _function.represent

类型：function

上下文访问：无关

表示一个函数。输出一个映射

- raw_input：是否不对输入求值
- code：函数调用过程的表示，`(ctx : input) : body`，`ctx` 为上下文变量名，`input` 为输入变量名，`body` 为函数体
- context：函数内上下文
- context_access：上下文访问权限
  - `_free`：无关
  - `_constant`：不变
  - `_mutable`：可变

## _function.apply

类型：function

上下文访问：可变

若 i 为 `func : input`，则以 input 为输入调用函数 func。

## _function.get_context_access

类型：function

上下文访问：不变

输出函数 c 的上下文访问权限。

- `_free`：无关
- `_constant`：不变
- `_mutable`：可变

## _function.get_raw_input

类型：function

上下文访问：不变

输出函数 c 是否不对输入求值。

## _function.is_primitive

类型：function

上下文访问：不变

输出函数 c 是否为基础函数。

## _function.get_code

类型：function

上下文访问：不变

输出函数 c 的代码的表示。

## _function.get_context

类型：function

上下文访问：不变

输出组合函数 c 的内上下文。
