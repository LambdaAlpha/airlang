# 控制

## .control.do

类型：function

上下文：可变

输入：相关

i 是列表，则按顺序执行列表 i 中的元素 `si`，输出最后一个元素求值的结果。

- 若 `si` 是一个调用 `_ .try i`，则对 `i` 求值，得到 `c`，若 `c` 为 `.(v)`，则不再继续对列表中之后的元素求值，输出 `v`
- 否则对 `si` 求值

## .control.test

类型：function

上下文：可变

输入：相关

若 i 为 `condition : branches`，`branches` 为 `branch1` 或者 `branch1 : branch2`

- 若 `condition` 为真，则执行 `branch1`，并输出其结果，执行方式参照 `do`
- 若 `condition` 为假且 `branch2` 存在，则执行 `branch2`，并输出其结果，执行方式参照 `do`

## .control.switch

类型：function

上下文：可变

输入：相关

若 i 为 `value : branches`

- 若 `branches` 是映射，则执行 `branches` 里 `value` 映射的值
- 若 `branches` 为 `map : default`
  - 若映射 `map` 中存在键 `value`，则执行其映射的值
  - 否则执行 `default`

## .control.match

类型：function

上下文：可变

输入：相关

若 i 为 `value : list`

- 逐个匹配列表项 `pattern : body` 的 `pattern` 与 `value`，若成功匹配，则执行绑定，并执行 `body`，输出其结果，执行方式参照 `do`

## .control.loop

类型：function

上下文：可变

输入：相关

若 i 为 `condition : body`

1. 执行 `condition`，得到 `c`
2. 若 `c` 不为真，则输出单元
3. 否则，执行 `body`，执行方式参照 `do`
   - 若执行结果为 `.(v)`，则输出 `v`
   - 否则转到 1

## .control.iterate

类型：function

上下文：可变

输入：相关

i 为 `value : name : body`，若 `value` 为以下类型，则迭代 `value`，将对应值绑定到 `name`，然后执行 `body`，执行方式参照 `do`，若执行结果为 `.(v)`，则输出 `v`，否则继续迭代。

- integer：若 `i` 大于等于 `0`，从 0（含）迭代到 `i`（不含）
- byte：迭代其每一个字节
- key：迭代其每一个字符
- text：迭代其每一个字符
- list：迭代其每一个项
- map：迭代其每一个键值对
