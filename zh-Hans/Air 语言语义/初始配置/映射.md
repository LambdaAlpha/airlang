# 映射

## .map.get_length

类型：function

上下文：不变

输入：无关

输出映射 c 的长度。

## .map.get_items

类型：function

上下文：不变

输入：无关

输出映射 c 的键值对的列表。

## .map.into_items

类型：function

上下文：可变

输入：无关

移除并输出映射 c 的键值对的列表。

## .map.get_keys

类型：function

上下文：不变

输入：无关

输出映射 c 的键的列表。

## .map.into_keys

类型：function

上下文：可变

输入：无关

移除并输出映射 c 的键的列表。

## .map.get_values

类型：function

上下文：不变

输入：无关

输出映射 c 的值的列表。

## .map.into_values

类型：function

上下文：可变

输入：无关

移除并输出映射 c 的值的列表。

## .map.contain

类型：function

上下文：不变

输入：相关

输出映射 c 是否包含键 i。

## .map.contain_all

类型：function

上下文：不变

输入：相关

输出映射 c 是否包含列表 i 中的所有键。

## .map.contain_any

类型：function

上下文：不变

输入：相关

输出映射 c 是否包含列表 i 中的任意键。

## .map.set

类型：function

上下文：可变

输入：相关

i 为 `k : v`，将映射 c 中键 `k` 所绑定的值设为 `v`，并输出其原来绑定的值的单元格。

## .map.set_many

类型：function

上下文：可变

输入：相关

将映射 i 中的键值对设置到映射 c 中，并输出这些键在 c 中原来绑定的值的映射。

## .map.get

类型：function

上下文：不变

输入：相关

输出映射 c 中键 i 所绑定的值的单元格。

## .map.get_many

类型：function

上下文：不变

输入：相关

输出键列表 i 中的键在映射 c 中所绑定的值的映射。

## .map.remove

类型：function

上下文：可变

输入：相关

移除并输出映射 c 中键 i 所绑定的值的单元格。

## .map.remove_many

类型：function

上下文：可变

输入：相关

移除并输出键列表 i 中的键在键 c 所绑定的映射中所绑定的值的映射。

## .map.move

类型：function

上下文：可变

输入：相关

移除并输出映射 c 中键 i 所绑定的值。

## .map.clear

类型：function

上下文：可变

输入：无关

清空映射 c。

## .map.make

类型：function

上下文：无关

输入：相关

从键值对列表 i 构建一个映射。

## .map.make_set

类型：function

上下文：无关

输入：相关

从键列表 i 构建一个映射，所有键绑定的值都是单元。
