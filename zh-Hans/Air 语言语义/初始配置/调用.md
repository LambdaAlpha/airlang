# 调用

## .call.make

类型：function

上下文：不变

输入：相关

i 为 `func : input`，输出调用 `_ func input`。

## .call.get_function

类型：function

上下文：不变

输入：无关

返回调用 c 的函数。

## .call.set_function

类型：function

上下文：可变

输入：相关

设置调用 c 的函数为 i，返回 c 原来的函数的值。

## .call.get_input

类型：function

上下文：不变

输入：无关

返回调用 `c` 的输入。

## .call.set_input

类型：function

上下文：可变

输入：相关

设置调用 c 的输入为 i，返回 c 原来的输入的值。
