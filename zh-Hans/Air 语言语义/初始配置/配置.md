# 配置

## _config.new

类型：function

上下文访问：无关

使用映射 i 构造一个配置。

## _config.represent

类型：function

上下文访问：无关

将配置 i 表示成一个映射。

## _config.exist

类型：function

上下文访问：无关

cfg 中是否存在键 i。

## _config.import

类型：function

上下文访问：无关

导入 cfg 中键 i 的值。

## _config.export

类型：function

上下文访问：无关

i 为 `key : value`，以 key 为键导出 value 到 cfg。

## _config.get_length

类型：function

上下文访问：不变

获取当前上下文配置的长度（数量）。

## _config.with

类型：function

上下文访问：可变

输入求值：否

i 为 `map : body`，用 map 中的键值对覆盖 cfg，执行 body，得到 o，恢复 cfg，返回 o。

## _config.self

类型：function

上下文访问：无关

获取当前配置。

## _config.where

类型：function

上下文访问：可变

输入求值：否

i 为 `cfg : body`，以 cfg 为配置，以 cfg 的初始上下文为上下文，执行 body。
