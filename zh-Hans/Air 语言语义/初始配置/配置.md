# 配置

## .config.make

类型：function

上下文：无关

输入：相关

使用映射 i 构造一个配置。

## .config.represent

类型：function

上下文：无关

输入：相关

将配置 i 表示成一个映射。

## .config.exist

类型：function

上下文：无关

输入：相关

cfg 中是否存在键 i。

## .config.import

类型：function

上下文：无关

输入：相关

导入 cfg 中键 i 的值。

## .config.export

类型：function

上下文：无关

输入：相关

i 为 `key : value`，以 key 为键导出 value 到 cfg。

## .config.get_length

类型：function

上下文：不变

输入：无关

获取当前上下文配置的长度（数量）。

## .config.with

类型：function

上下文：可变

输入：相关

i 为 `map : body`，用 map 中的键值对覆盖 cfg，执行 body，得到 o，恢复 cfg，返回 o。

## .config.get_self

类型：function

上下文：无关

输入：无关

获取当前配置。

## .config.where

类型：function

上下文：可变

输入：相关

i 为 `cfg : body`，在 c 中以键 cfg 为键，得到上下文 c1，以 c1 为配置，以 c1 的初始上下文为上下文，执行 body。
