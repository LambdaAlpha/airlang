# 字符串

## string.from_utf8

类型：function

上下文访问模式：无关

将合法的 utf8 编码的字节 input 转化为字符串。

## string.into_utf8

类型：function

上下文访问模式：无关

将字符串 input 按 utf8 编码转化为字节。

## string.length

类型：function

上下文访问模式：不变

输入模式：`{symbol}`

输出字符串 input 的长度。

## string.push

类型：function

上下文访问模式：可变

输入模式：`{symbol} : .t`

input 为 `s1 : s2`，将字符串 `s2` 附加到符号 `s1` 所绑定的字符串的后面。

## string.concat

类型：function

上下文访问模式：无关

输入模式：`{list : .t}`

将字符串列表 input 拼接成一个字符串。
