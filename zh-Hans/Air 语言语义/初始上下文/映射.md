# 映射

## map.length

类型：function

上下文访问模式：不变

输入模式：`{symbol}`

输出符号 input 所绑定的映射的长度。

## map.items

类型：function

上下文访问模式：不变

输入模式：`{symbol}`

输出模式：`{list : (.true all .true)}`

输出符号 input 所绑定的映射的键值对的列表。

## map.into_items

类型：function

上下文访问模式：可变

输入模式：`{symbol}`

输出模式：`{list : (.true all .true)}`

移除并输出符号 input 所绑定的映射的键值对的列表。

## map.keys

类型：function

上下文访问模式：不变

输入模式：`{symbol}`

输出模式：`{list : .true}`

输出符号 input 所绑定的映射的键的列表。

## map.into_keys

类型：function

上下文访问模式：可变

输入模式：`{symbol}`

输出模式：`{list : .true}`

移除并输出符号 input 所绑定的映射的键的列表。

## map.values

类型：function

上下文访问模式：不变

输入模式：`{symbol}`

输出模式：`{list : .true}`

输出符号 input 所绑定的映射的值的列表。

## map.into_values

类型：function

上下文访问模式：可变

输入模式：`{symbol}`

输出模式：`{list : .true}`

移除并输出符号 input 所绑定的映射的值的列表。

## map.contains

类型：function

上下文访问模式：不变

输入模式：`{symbol} : .true`

input 为 `n : k`，输出符号 `n` 所绑定的映射是否包含键 `k`。

## map.contains_many

类型：function

上下文访问模式：不变

输入模式：`{symbol} : {list : .true}`

input 为 `n : keys`，输出符号 `n` 所绑定的映射是否包含列表 `keys` 中的所有键。

## map.set

类型：function

上下文访问模式：可变

输入模式：`{symbol} : (: .true)`

input 为 `n : (k : v)`，将符号 `n` 所绑定的映射中键 `k` 所绑定的值设为 `v`，并输出其原来绑定的值。

## map.set_many

类型：function

上下文访问模式：可变

输入模式：`{symbol} : {map : .true}`

input 为 `n : map`，将 `map` 中的键值对设置到符号 `n` 所绑定的映射中，并输出这些键在 `n` 中原来绑定的值的映射。

## map.get

类型：function

上下文访问模式：不变

输入模式：`{symbol} : .true`

input 为 `n : k`，输出符号 `n` 所绑定的映射中键 `k` 所绑定的值。

## map.get_many

类型：function

上下文访问模式：不变

输入模式：`{symbol} : {list : .true}`

input 为 `n : keys`，输出符号列表 `keys` 中的键在符号 `n` 所绑定的映射中所绑定的值的映射。

## map.remove

类型：function

上下文访问模式：可变

输入模式：`{symbol} : .true`

input 为 `n : k`，移除并输出符号 `n` 所绑定的映射中键 `k` 所绑定的值。

## map.remove_many

类型：function

上下文访问模式：可变

输入模式：`{symbol} : {list : .true}`

input 为 `n : keys`，移除并输出符号列表 `keys` 中的键在符号 `n` 所绑定的映射中所绑定的值的映射。

## map.clear

类型：function

上下文访问模式：可变

输入模式：`{symbol} : .true`

清空符号 `n` 所绑定的映射。

## map

类型：function

上下文访问模式：无关

输入模式：`{list : (.true all .true)}`

输出模式：`{map : .true}`

从键值对列表 input 构建一个映射。

## set

类型：function

上下文访问模式：无关

输入模式：`{list : .true}`

输出模式：`{map : .true}`

从键列表 input 构建一个映射，所有键绑定的值都是单元。

## multiset

类型：function

上下文访问模式：无关

输入模式：`{list : .true}`

输出模式：`{map : .true}`

从值列表 input 构建一个映射，键是值列表中的值，值是其在值列表中出现的次数。
