# 函数

## function

类型：function

上下文访问模式：无关

调用模式：`{map : {body : form, prelude : eval, input_name : eval, context_name : eval, context_access : eval, call_mode : form, ask_mode : form, cacheable : eval, static : eval}}`

构建一个组合函数。

i 是一个映射

- body：函数体
- prelude：函数内置的上下文
- input_name：输入的名称
- context_name：调用上下文的名称
- context_access：上下文访问模式
  - free：无关
  - constant：不变
  - mutable：可变
- call_mode：调用模式
- ask_mode：询问模式
- cacheable：可缓存性
- static：静态

## function.represent

- body：函数体

- input_name：输入的名称
- context_name：调用上下文的名称
- context_access：上下文访问模式
  - free：无关
  - constant：不变
  - mutable：可变
- call_mode：调用模式
- ask_mode：询问模式
- id：基础函数的 id
- is_extension：基础函数是否是拓展函数
- static：静态
