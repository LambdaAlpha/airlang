# 函数

## function

类型：function

上下文访问：无关

调用输入模式：`{map : {code : form@, context_access : {symbol : '.'}} : eval@}`

构造一个函数。

i 是一个映射

- id：基础函数的 id
- code：函数调用过程的表示，`(ctx : input) : body`，`ctx` 为上下文变量名，`input` 为输入变量名，`body` 为函数体
- context：函数内上下文
- context_access：上下文访问权限
  - free：无关
  - constant：不变
  - mutable：可变
- call_setup：调用设置
- solve_setup：求解设置
- cell：细胞
