## function

类型：function

上下文访问：无关

调用模式：`{map : {call : {pair : {symbol : ..} : {change : {symbol : ..} : form*}, change : {symbol : ..} : form*}, context_access : {symbol : ..}}}`

构造一个组合函数。

i 是一个映射

- call：函数调用过程的表示，`ctx : input -> body`，`ctx` 为上下文变量名，`input` 为输入变量名，`body` 为函数体
- context：函数内上下文
- context_access：上下文访问权限
  - free：无关
  - constant：不变
  - mutable：可变
- call_mode：调用模式
- optimize_mode：优化模式
- solve_mode：解决模式
- static：静态

## function.represent

类型：function

上下文访问：无关

解决模式：`{map : {call : {pair : {symbol : ..} : {change : {symbol : ..} : form*}, change : {symbol : ..} : form*}, context_access : {symbol : ..}}}`

表示一个函数。输出一个映射

- call：函数调用过程的表示，`ctx : input -> body`，`ctx` 为上下文变量名，`input` 为输入变量名，`body` 为函数体
- context：函数内上下文
- context_access：上下文访问权限
  - free：无关
  - constant：不变
  - mutable：可变
- call_mode：调用模式
- optimize_mode：优化模式
- solve_mode：解决模式
- id：基础函数的 id
- is_extension：基础函数是否是拓展函数
- static：静态
