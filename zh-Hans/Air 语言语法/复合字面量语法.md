# 复合字面量语法

## 配对

`a : b`

示例：

```air
a : b
a : []
[] : a
a : b : c
(a : b) : c
a : (b c)
(a b) : c
a : (b [])
(a []) : b
a : b ? c
(a ? b) : c
a : b c d
(a b c) : d
```

## 调用

`a ! b`

示例：

```Air
a ! b
a ! b ! c
a ! []
a ! b : c
```

## 询问

`a ? b`

示例：

```air
a ? b
a ? []
[] ? a
a ? (b c)
(a b) ? c
a ? (b [])
(a []) ? b
a ? b ? c
(a ? b) ? c
a ? b c d
(a b c) ? d
```

## 注释

`meta ; value`

`meta` 和 `value` 为任意字面量。

注释用于表达元信息。

示例：

```air
a ; 1
"this is a comment" ; .
1 ; 2
(a []) ; .
```

## 列表

`[a, b, ..., z]` 或 `[a, b, ..., z,]`

示例：

```air
[]
[a]
[a,]
[a, b]
[a, b,]
[a, b, c]
[a, b, c,]
```

## 映射

`{a : b, ..., y : z}` 或 `{a : b, ..., y : z,}`

其中键为原子结构

示例：

```air
{}
{a : b}
{a : b,}
{a : b, c : d}
{a : b, c : d,}
{a : b, c : d, e : f}
{a : b, c : d, e : f,}
{a}
{a,}
{a, b}
{a, b,}
{a, b : c}
{a : b, c}
```
