# 复合字面量语法

## 配对

`a : b`

示例：

```air
a : b
a : []
[] : a
a : b : c
(a : b) : c
a : (b c)
(a b) : c
a : (b [])
(a []) : b
a : _ b c
(_ a b) : c
a : b c d
(a b c) : d
```

## 调用

- `_ f a`：`f` 为函数，`a` 为输入
- `a f _`：`f` 为函数，`a` 为输入
- `a f b`：`f` 为函数，`a : b` 为输入

示例：

```Air
_ a b
_ a _ b c
_ a []
_ a b : c
a f b
a f b g c
a f _ g b
_ f a g b
_ f _ g a
```

## 列表

`[a, b, ..., z]` 或 `[a, b, ..., z,]`

`_[a b ... z]`

示例：

```air
[]
[a]
[a,]
[a, b]
[a, b,]
[a, b, c]
[a, b, c,]
_[]
_[a]
_[a b]
_[a b c]
```

## 映射

`{a1 : a2, ..., z1 : z2}` 或 `{a1 : a2, ..., z1 : z2,}`

`_{a1 a2 ... z1 z2}`

其中 `x1` 为键。一个映射中的键不能重复。

如果键值对中的值为单元，则可以简写，比如 `{a : ., b : c}` 可以简写为 `{a, b : c}`。

示例：

```air
{}
{a : b}
{a : b,}
{a : b, c : d}
{a : b, c : d,}
{a : b, c : d, e : f}
{a : b, c : d, e : f,}
{a}
{a,}
{a, b}
{a, b,}
{a, b : c}
{a : b, c}
_{}
_{a b}
_{a b c d}
_{
a b
c d
e f
}
```
